# Q-Learning Tutorial

A simple tabular-based Q-learning tutorial for CartPole-v0 in gym

## Requirements

- python >= 3.6
- OpenAI Gym
- numpy
- matplotlib

## Run Python Code

```
#Training
python train.py

#Evaluation
python test.py
```
